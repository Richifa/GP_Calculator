Simple GPA Calculator App

This Java application is a straightforward GPA (Grade Point Average) calculator designed to help students calculate their cumulative GPA based on individual course grades and credit hours. The application is built with simplicity and ease of use in mind, making it suitable for students at various levels of familiarity with programming.

Features

  • User-Friendly Interface: The application provides a simple command-line interface for entering course details and obtaining the cumulative GPA.
  • Grade and Credit Entry: Users can input their course grades and corresponding credit hours, allowing for accurate GPA calculation.
  • Cumulative GPA Calculation: The app calculates the cumulative GPA based on the entered course grades and credit hours.

How to Use

  1. Clone the Repository:
    2. git clone https://github.com/your-username/GPACalculator.git
cd GPACalculator
  2. Compile the Java Files:
    3. javac GPACalculator.java
  3. Run the Application:
    4. java GPACalculator
  4. Follow On-Screen Instructions:
    • Enter the number of courses.
    • For each course, input the grade and credit hours.
  5. View Cumulative GPA:
    • The application will display the cumulative GPA based on the entered information.
